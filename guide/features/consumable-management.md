# 耗材管理

## 耗材统计

提供库存数据的可视化分析，包括：库存总览、分类统计、库存预警、领用趋势。

## 耗材列表

### 新增耗材

1. 进入「耗材列表」页面
2. 点击「新增耗材」
3. 填写耗材信息：名称、分类、规格、计量单位、库存数量、安全库存等
4. 保存

### 库存操作

| 操作 | 说明 |
|------|------|
| **入库** | 增加库存数量 |
| **出库/领用** | 减少库存数量，记录领用人 |
| **盘点** | 更新实际库存数量 |

### SN 序列号追踪

对于需要精确管理的耗材（如硬盘、内存模块等），支持 SN 序列号级别的追踪管理：

- **SN 录入**：入库时逐个输入或批量扫描耗材的 SN 序列号
- **SN 追溯**：在耗材详情页可查看每个 SN 序列号的完整流转记录（入库时间、使用去向、当前状态等）
- **SN 出库**：出库/领用时扫描具体 SN，确保实物与系统记录一致
- **SN 状态标识**：
  - 🟢 在库：序列号对应的耗材在库存中
  - 🟡 已领用：序列号对应的耗材已出库分配给指定人员
  - 🔴 已报废：序列号对应的耗材已报废处理

### 版本号乐观锁

耗材模块采用乐观锁机制确保并发操作下的数据一致性：

- **实现方式**：每条耗材记录维护一个版本号（version 字段）
- **更新校验**：执行入库、出库、盘点等操作时，系统校验当前版本号与操作发起时的版本号是否一致
- **冲突处理**：若版本号不一致（表示有其他用户同时修改了该记录），系统提示"数据已被其他操作修改，请刷新后重试"
- **适用范围**：所有修改库存数量的操作均受乐观锁保护，避免多用户同时操作导致的库存数量不一致问题

## 分类管理

分类管理用于维护耗材的分类体系，支持多级分类。

## 操作日志

记录所有耗材的出入库操作历史，支持按时间范围、耗材名称、操作类型筛选查询。

### 日志归档

操作日志支持自动归档机制，以保障系统性能和满足合规要求：

- **归档策略**：系统默认保留近 12 个月的操作日志，超出部分自动归档
- **归档存储**：归档日志以压缩格式存储，支持导出为 Excel/CSV 文件本地保存
- **归档查询**：归档后的日志仍可在系统中查询，但加载速度低于实时日志
- **手动归档**：管理员可随时手动触发归档操作，并指定归档的时间范围
- **归档清理**：归档超过 36 个月的日志系统自动清理，释放存储空间

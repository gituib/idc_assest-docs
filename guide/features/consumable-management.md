# 耗材管理

## 耗材统计

提供库存数据的可视化分析，包括：库存总览、分类统计、库存预警、领用趋势。

## 耗材列表

### 新增耗材

1. 进入「耗材列表」页面
2. 点击「新增耗材」
3. 填写耗材信息：名称、分类、规格、计量单位、库存数量、安全库存等
4. 保存

### 库存操作

| 操作 | 说明 |
|------|------|
| **入库** | 增加库存数量 |
| **出库/领用** | 减少库存数量，记录领用人 |
| **盘点** | 更新实际库存数量 |

## 分类管理

分类管理用于维护耗材的分类体系，支持多级分类。

## 操作日志

记录所有耗材的出入库操作历史，支持按时间范围、耗材名称、操作类型筛选查询。

# 系统管理

## 用户管理

用户管理用于管理系统用户账号和权限。

### 用户状态

| 状态 | 说明 |
|------|------|
| **正常** | 用户可正常登录 |
| **待审核** | 新注册用户，等待管理员审核 |
| **已禁用** | 账号被禁用，无法登录 |

### 角色权限

系统采用 RBAC 权限模型：

| 角色 | 权限范围 |
|------|----------|
| **管理员** | 全部功能 |
| **运维人员** | 设备管理、工单处理 |
| **普通用户** | 查看权限 |

## 系统设置

| 设置项 | 说明 |
|--------|------|
| **站点名称** | 系统显示名称 |
| **Logo** | 系统Logo图片 |
| **空闲超时** | 无操作自动退出时间 |
| **背景图片** | 登录页背景 |

# 机房管理

## 机房管理

机房管理用于维护数据中心的基础信息。

### 新增机房

1. 进入「机房管理」页面
2. 点击「新增机房」按钮
3. 填写机房信息：机房名称、位置、面积、负责人、联系电话、备注、机房图片
4. 点击「确定」保存

### 删除机房

::: warning 注意
删除机房前需确保该机房下没有机柜和设备。
:::

## 机柜管理

### 新增机柜

1. 进入「机柜管理」页面
2. 点击「新增机柜」按钮
3. 填写机柜信息：机柜名称、所属机房、机柜高度(U数)、位置
4. 点击「确定」保存

### 容量查看

机柜列表显示：总U数、已用U数、剩余U数、使用率（可视化进度条）。

## 3D机柜可视化

3D可视化提供直观的机柜设备布局展示。

### 操作方式

| 操作 | 方法 |
|------|------|
| **旋转视角** | 鼠标左键拖拽 |
| **缩放视图** | 鼠标滚轮 |
| **平移视图** | 鼠标右键拖拽 |
| **查看设备** | 鼠标悬停显示设备信息 |
| **设备详情** | 点击设备弹出详情面板 |

### 设备状态颜色

- 🟢 绿色：运行中
- 🟡 黄色：维护中
- 🔴 红色：故障
- ⚪ 灰色：已停用

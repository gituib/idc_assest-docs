# 机房管理

## 机房管理

机房管理用于维护数据中心的基础信息。

### 新增机房

1. 进入「机房管理」页面
2. 点击「新增机房」按钮
3. 填写机房信息：机房名称、位置、面积、负责人、联系电话、备注、机房图片
4. 点击「确定」保存

### 删除机房

::: warning 注意
删除机房前需确保该机房下没有机柜和设备。
:::

### 机房平面图

机房平面图以 2D 俯视图方式展示机房的物理布局，帮助运维人员直观掌握机房空间分配和设备部署情况。

#### 展示内容

- **机柜分布**：在平面图上标注每个机柜的位置，显示机柜编号
- **区域划分**：支持在平面图中标记不同功能区域（如服务器区、网络区、存储区、配电区等）
- **通道标识**：标注冷通道、热通道位置及方向
- **基础设施标注**：标注空调、配电柜、UPS、消防设备等基础设施位置

#### 操作方式

| 操作 | 方法 |
|------|------|
| **缩放平面图** | 鼠标滚轮缩放 |
| **平移视图** | 鼠标拖拽空白区域 |
| **查看机柜详情** | 点击机柜位弹出信息面板 |
| **查看设备统计** | 鼠标悬停显示该区域设备数量统计 |

#### 编辑平面图

1. 进入机房详情页 → 「平面图」标签页
2. 点击「编辑平面图」按钮
3. 可拖拽调整机柜位置，添加或删除区域标注
4. 编辑完成后保存

## 机柜管理

### 新增机柜

1. 进入「机柜管理」页面
2. 点击「新增机柜」按钮
3. 填写机柜信息：机柜名称、所属机房、机柜高度(U数)、位置
4. 点击「确定」保存

### 容量查看

机柜列表显示：总U数、已用U数、剩余U数、使用率（可视化进度条）。

### 批量导入导出机柜

#### 批量导入

1. 点击「导入」按钮
2. 下载导入模板（Excel/CSV 格式），模板包含：机柜名称、所属机房、机柜高度（U数）、位置编码等列
3. 按模板格式填写多个机柜信息
4. 上传文件，系统自动解析并批量创建机柜
5. 导入完成后显示成功与失败记录详情

#### 批量导出

1. 在机柜列表设置筛选条件（按机房、按状态等）
2. 点击「导出」按钮
3. 选择导出格式（Excel/CSV）
4. 导出的清单包含：机柜名称、所属机房、总U数、已用U数、使用率、位置编码等信息

## 3D机柜可视化

3D可视化提供直观的机柜设备布局展示，基于三维渲染引擎实现。

### 视图能力

- **设备排布**：在 3D 机柜模型中展示设备的实际安装位置，精确到 U 位
- **设备面板**：渲染设备前面板，展示指示灯、端口等外观特征
- **空间感**：通过三维视角直观展示机柜的立体空间和设备占用情况
- **多机柜视图**：支持切换到整排机柜或整个机房的 3D 概览视图

### 操作方式

| 操作 | 方法 |
|------|------|
| **旋转视角** | 鼠标左键拖拽 |
| **缩放视图** | 鼠标滚轮 |
| **平移视图** | 鼠标右键拖拽 |
| **查看设备** | 鼠标悬停显示设备信息 |
| **设备详情** | 点击设备弹出详情面板 |

### 设备状态颜色

| 颜色 | 对应状态 |
|------|----------|
| 🟢 **绿色** | 运行中 — 设备正常运行 |
| 🟡 **黄色** | 维护中 — 设备正在维护 |
| 🔴 **红色** | 故障 — 设备出现异常 |
| ⚪ **灰色** | 已停用/已报废 — 设备不再使用 |

### 筛选与定位

- 支持按设备名称、类型快速搜索定位
- 搜索命中后，3D 视图自动聚焦到目标设备并高亮显示
- 支持按设备状态筛选显示

用户管理 ​

用户管理用于管理系统用户账号和权限。

用户状态 ​

状态说明
正常用户可正常登录系统并使用相应权限功能
待审核新注册用户，等待管理员审核通过后方可登录
已禁用账号被管理员禁用，无法登录系统

审核流程 ​

用户注册审核流程如下：

  1. 用户注册：新用户通过注册页面提交账号信息
  2. 待审核状态：注册成功后，账号自动进入「待审核」状态
  3. 管理员审核：管理员在「用户管理」页面查看待审核用户列表
  4. 审核操作：管理员可「通过」或「驳回」注册申请，驳回需填写原因
  5. 结果通知：
    • 审核通过：用户收到通知，账号状态变更为「正常」，可登录系统
    • 审核驳回：用户收到通知及驳回原因，可根据原因修改后重新提交注册

角色权限 ​

系统采用 RBAC（基于角色的访问控制）权限模型：

角色权限范围
管理员全部功能，包括系统设置、用户管理、数据管理、所有模块的增删改查
运维人员设备管理（含导入导出、端口管理、接线管理）、工单处理（接单、处理、完成）
普通用户查看权限（查看设备、工单、仪表盘等），可创建工单

角色管理 ​

  • 自定义角色：支持管理员自定义角色，为不同岗位配置专属权限组合
  • 权限粒度：权限控制到页面级和操作级（新增、编辑、删除、查看、导入、导出）
  • 角色分配：在用户编辑页面勾选角色，支持一个用户拥有多个角色

系统设置 ​

系统设置用于配置系统全局参数和外观。

设置项 ​

设置项说明
站点名称系统浏览器标签页标题和页面左上角显示的名称
Logo系统左上角显示的 Logo 图片，建议尺寸 120×40px，支持 PNG/SVG 格式
空闲超时用户无操作后自动退出登录的时间，可选范围：15分钟、30分钟、60分钟、永不超时
背景图片登录页的背景图片，支持 JPG/PNG 格式
维护模式开启后系统仅管理员可登录，其他用户访问时显示维护提示页面
登录限制可限制指定 IP 段或仅允许内网 IP 访问系统登录页

操作说明 ​

  1. 进入「系统管理」→「系统设置」页面
  2. 修改配置项后点击「保存」按钮
  3. 部分设置（如站点名称、Logo、空闲超时）保存后立即生效
  4. 维护模式和登录限制等安全设置保存后，下一次登录时生效

# 仪表盘

仪表盘是系统的首页，提供数据中心整体运行状态的可视化展示。

## 统计卡片

| 区域 | 内容 |
|------|------|
| **统计卡片** | 机房数量、机柜数量、设备总数、运行中设备、维护中设备、已报废设备 |
| **设备状态分布** | 各状态设备占比图表 |
| **机柜容量概览** | 各机房机柜使用率 |
| **待处理工单** | 未完成工单列表 |
| **库存预警** | 低库存耗材提醒 |

## 功率监控

功率监控模块展示数据中心的电力负载情况，帮助运维人员掌握能耗分布，预防过载风险。

### 监控指标

| 指标 | 说明 |
|------|------|
| **总功率** | 当前数据中心所有设备的总功率消耗（kW） |
| **机柜功率** | 各机柜的实时功率消耗及功率密度分布 |
| **功率使用效率（PUE）** | 数据中心总能耗与 IT 设备能耗的比值 |
| **功率阈值告警** | 当机柜功率超过设定阈值时触发告警提示 |

### 展示方式

- **机柜功率热力图**：以热力图方式展示各机柜的功率负载，颜色从绿到红渐变，直观反映功率分布
- **功率排行**：按功率消耗排序的设备/机柜 TOP 列表，便于定位高能耗单元
- **功率趋势**：展示选定时间范围内的功率变化曲线

## 趋势图表

趋势图表模块提供关键指标的历史变化趋势分析，支持不同时间粒度的数据查看。

### 趋势类型

| 趋势类型 | 展示内容 |
|----------|----------|
| **设备数量趋势** | 按日/周/月展示设备总数及各状态设备数量变化曲线 |
| **功率趋势** | 按时间维度展示总功率、机柜功率的变化曲线 |
| **工单趋势** | 按日/周/月展示工单创建量、完成量的变化趋势 |
| **耗材领用趋势** | 展示耗材出入库数量的时间序列变化 |

### 操作方式

- **时间粒度切换**：支持按日、按周、按月、按年切换统计粒度
- **时间范围选择**：支持自定义起止时间，以及近 7 天、近 30 天、近 90 天等快捷选项
- **图例交互**：点击图例可显示/隐藏对应数据系列
- **数据悬浮**：鼠标悬停在图表数据点上，显示具体数值和日期

## 机柜容量分布图

机柜容量分布图以可视化方式展示各机柜的空间使用情况，辅助运维人员快速定位可用资源。

### 展示内容

- **机柜使用率**：以进度条或百分比形式展示每个机柜的 U 位使用率
- **分布概览**：按机房维度聚合展示机柜使用率分布状况
- **颜色标识**：
  - 🟢 绿色：使用率低于 60%，资源充足
  - 🟡 黄色：使用率 60%～85%，资源适中
  - 🔴 红色：使用率高于 85%，资源紧张
- **空闲 U 位统计**：各机房剩余可用 U 位的汇总数据

### 交互操作

- 点击机柜可跳转至该机柜的设备列表详情页
- 支持按机房切换查看不同区域的机柜容量
- 支持按使用率排序，优先展示资源紧张或资源充足的机柜

## 数据刷新机制

仪表盘数据支持多种刷新方式，确保展示内容的实时性和准确性。

### 手动刷新

点击仪表盘右上角的「刷新」按钮，立即触发全量数据重新加载。

### 自动刷新

- **默认间隔**：系统每 30 秒自动轮询更新数据
- **刷新范围**：自动刷新仅更新统计卡片数值等轻量数据，趋势图表和分布图在切换时间粒度时按需加载
- **后台静默**：自动刷新在后台进行，不影响用户当前操作

### 刷新状态提示

- 数据加载中时，相关卡片区域显示加载动画
- 数据更新成功后，卡片区域有轻量闪烁提示
- 数据加载失败时显示错误提示，并保留上次成功加载的数据

## 操作说明

- **快速跳转**：点击统计卡片可跳转到对应管理页面

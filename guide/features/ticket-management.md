# 工单管理

## 工单列表

### 工单状态

```
待处理 → 处理中 → 已完成 → 已关闭
```

| 状态 | 说明 |
|------|------|
| **待处理** | 新建工单，等待分配 |
| **处理中** | 正在处理 |
| **已完成** | 问题已解决 |
| **已关闭** | 工单关闭 |

### 工单优先级

| 优先级 | 说明 | 响应时间 |
|--------|------|----------|
| **紧急** | 严重影响业务 | 立即响应 |
| **高** | 重要问题 | 2小时内 |
| **中** | 一般问题 | 24小时内 |
| **低** | 优化建议 | 一周内 |

### 创建工单

1. 进入「工单列表」页面
2. 点击「新建工单」
3. 填写工单信息：标题、故障分类、优先级、关联设备、问题描述、附件等
4. 提交工单

### 处理工单

1. 点击工单进入详情页
2. 点击「接单」分配给自己
3. 更新工单状态为「处理中」
4. 记录处理过程
5. 问题解决后，点击「完成」

## 故障分类

用于标准化故障类型，便于统计分析。支持硬件故障、网络故障、系统故障等分类。

## 统计报表

提供工单数量趋势、分类统计、处理效率、人员统计等分析视图。

## 字段管理

工单字段管理用于自定义工单属性，操作方式与设备字段管理相同。

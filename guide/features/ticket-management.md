# 工单管理

## 工单列表

### 工单状态

```
待处理 → 处理中 → 已完成 → 已关闭
```

| 状态 | 说明 |
|------|------|
| **待处理** | 新建工单，等待分配 |
| **处理中** | 正在处理 |
| **已完成** | 问题已解决 |
| **已关闭** | 工单关闭 |

### 工单优先级

| 优先级 | 说明 | 响应时间 |
|--------|------|----------|
| **紧急** | 严重影响业务 | 立即响应 |
| **高** | 重要问题 | 2小时内 |
| **中** | 一般问题 | 24小时内 |
| **低** | 优化建议 | 一周内 |

### 创建工单

1. 进入「工单列表」页面
2. 点击「新建工单」
3. 填写工单信息：标题、故障分类、优先级、关联设备、问题描述、附件等
4. 提交工单

### 处理工单

1. 点击工单进入详情页
2. 点击「接单」分配给自己
3. 更新工单状态为「处理中」
4. 记录处理过程
5. 问题解决后，点击「完成」

### 工单关联设备

在创建或编辑工单时，支持将工单与一个或多个设备进行关联：

- **关联方式**：在工单表单中点击「选择设备」按钮，从设备列表中勾选目标设备
- **关联展示**：工单详情页顶部展示关联设备列表，点击设备名称可快速跳转至设备详情页
- **设备工单追溯**：在设备详情页的「工单历史」标签页中，可查看与该设备相关的所有历史工单
- **关联解除**：在工单详情页点击设备列表中的「解除关联」按钮，可移除工单与设备的关联关系

通过设备关联，运维人员可在处理设备故障时快速了解该设备的历史问题，辅助故障诊断。

## 故障分类

故障分类用于标准化故障类型，便于后续统计分析和运维流程优化。

### 分类层级

支持多级分类结构，默认提供以下一级分类：

| 分类 | 说明 | 示例子类 |
|------|------|----------|
| **硬件故障** | 设备硬件相关问题 | 硬盘故障、内存故障、电源故障、风扇故障、主板故障 |
| **网络故障** | 网络连接相关问题 | 端口 Down、链路中断、带宽异常、DNS 解析失败 |
| **系统故障** | 操作系统或软件问题 | 系统崩溃、服务异常、进程挂死、配置错误 |
| **环境故障** | 机房环境相关问题 | 温度过高、湿度异常、漏水告警、供电异常 |
| **安全事件** | 安全相关告警 | 异常登录、病毒告警、端口扫描、DDoS 攻击 |
| **其他** | 以上未涵盖的故障类型 | — |

### 管理故障分类

1. 进入「系统管理」→「故障分类配置」页面
2. 支持新增、编辑、删除故障分类
3. 支持拖拽排序和父子层级管理
4. 修改后的分类结构将同步到工单创建表单的故障分类下拉选项

## 统计报表

统计报表模块提供工单数据的多维度分析，辅助管理决策。

### 报表视图

| 报表类型 | 说明 |
|----------|------|
| **工单数量趋势** | 按日/周/月展示工单创建量的变化趋势，折线图展示 |
| **分类统计** | 按故障分类统计工单数量，饼图/柱状图展示各类占比 |
| **处理效率** | 统计工单平均响应时间、平均处理时长、按时完成率等效率指标 |
| **人员统计** | 按处理人统计工单数量、完成率、平均处理时长 |
| **优先级分布** | 按优先级展示工单数量分布 |
| **状态分布** | 按工单状态（待处理、处理中、已完成、已关闭）统计数量 |

### 报表筛选

- 支持按时间范围筛选（自定义起止时间、预设快捷时间如近 7 天、近 30 天、近 90 天）
- 支持按故障分类筛选
- 支持按工单优先级筛选
- 支持按处理人筛选

### 报表导出

统计报表支持导出为图片（PNG）或数据表格（Excel），方便用于运维报告和复盘分析。

## 字段管理

工单字段管理用于自定义工单属性，满足不同运维场景下的信息收集需求。

### 支持字段类型

| 字段类型 | 说明 |
|----------|------|
| **文本** | 单行文本输入 |
| **多行文本** | 长文本输入 |
| **数字** | 数值输入 |
| **日期** | 日期选择器 |
| **日期时间** | 日期+时间选择器 |
| **下拉选择** | 单选下拉框，可自定义选项列表 |
| **多选** | 多选下拉框，可同时选择多个选项 |
| **开关** | 布尔值开关 |
| **附件** | 文件上传 |

### 管理自定义字段

1. 进入「工单管理」页面 → 「字段管理」标签页
2. 点击「新增字段」按钮
3. 填写字段信息：字段名称、字段类型、是否必填、排序、默认值、选项列表（下拉/多选类型需要）
4. 保存后，自定义字段将自动出现在工单创建和编辑表单中

操作方式与设备字段管理保持一致，确保用户在不同模块间有一致的操作体验。

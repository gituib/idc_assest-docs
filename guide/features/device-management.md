# 资产管理

## 设备管理

设备管理是系统的核心功能，提供设备全生命周期管理。

### 设备状态流转

```
采购中 → 待安装 → 运行中 ⇄ 维护中 → 已报废
```

| 状态 | 说明 |
|------|------|
| **采购中** | 设备已采购，尚未到货 |
| **待安装** | 设备到货，等待安装 |
| **运行中** | 设备正常运行 |
| **维护中** | 设备正在维护 |
| **已报废** | 设备已报废处理 |

### 新增设备

1. 进入「设备管理」页面
2. 点击「新增设备」按钮
3. 填写设备信息：名称、类型、型号、序列号、品牌、所属机房、机柜、U位、状态等
4. 点击「确定」保存

### 批量导入

1. 点击「导入」按钮
2. 下载导入模板（Excel/CSV格式）
3. 按模板格式填写设备信息
4. 上传文件，系统自动解析并导入

### 设备导出

1. 选择需要导出的设备（可多选）
2. 点击「导出」按钮
3. 选择导出格式（Excel/CSV）
4. 文件自动下载

### 批量操作

系统支持对多台设备进行批量管理，提升运维效率：

| 操作 | 说明 |
|------|------|
| **批量修改** | 选中多台设备，批量修改机房、机柜、状态、负责人等公共属性 |
| **批量删除** | 选中多台设备，批量删除不再管理的设备记录 |
| **批量导出** | 按筛选条件或手动选择范围，导出为 Excel 或 CSV 格式 |
| **批量移动** | 将选中的设备批量迁移至指定机柜或机房 |

操作方式：在设备列表勾选目标设备，点击列表上方的批量操作按钮，在弹出的对话框中填写修改内容后确认提交。

## 端口管理

端口管理用于配置设备的网络端口信息，是网络运维的基础数据支撑。

> **v2.1.1 更新**：端口管理页面已改为**按设备分组分页**展示，解决了端口数量过多时部分设备不显示的问题。端口名称统一使用自然排序，展示顺序更直观。

### 端口类型

| 类型 | 说明 |
|------|------|
| **网口** | 以太网接口（RJ45），通常为 10/100/1000Mbps 自适应电口 |
| **光口** | 光纤接口（SFP/SFP+/QSFP），支持千兆/万兆/25G/40G/100G 传输速率 |
| **管理口** | 带外管理接口（如 iLO、iDRAC、BMC），用于远程管理设备 |
| **其他** | 其他类型接口（如 Console 口、USB 口、串口等） |

### 端口属性

每个端口包含以下信息：

| 属性 | 说明 |
|------|------|
| **端口名称** | 如 eth0、GigabitEthernet0/0/1 等 |
| **端口类型** | 网口/光口/管理口/其他 |
| **MAC地址** | 端口的物理地址 |
| **速率** | 端口协商速率（如 1000Mbps、10Gbps） |
| **绑定网卡** | 将该端口关联到已纳管的网卡设备 |
| **连接状态** | 已连接/未连接 |
| **连接对象** | 该端口所连接的交换机端口或对端设备端口 |
| **备注** | 端口用途描述等补充信息 |

### 绑定网卡

端口管理支持将端口与已记录的网卡设备进行绑定。操作路径：设备详情页 → 端口管理 → 点击指定端口的「绑定网卡」按钮 → 搜索并选择目标网卡 → 确认绑定。绑定后可在网卡详情页查看关联的物理端口信息。

### 批量管理端口

1. 在设备详情页进入「端口管理」标签页
2. 勾选多个端口
3. 点击「批量编辑」按钮
4. 可批量修改端口类型、速率、备注等属性
5. 确认后一次性保存所有修改

## 接线管理

接线管理用于记录机柜间、设备间的线缆连接关系，支持线缆可视化追踪和清单导出。

### 连接关系

接线管理记录完整的连接拓扑信息：

| 信息项 | 说明 |
|--------|------|
| **源设备** | 线缆起始端设备 |
| **源端口** | 起始端设备的具体端口 |
| **目标设备** | 线缆终止端设备 |
| **目标端口** | 终止端设备的具体端口 |
| **线缆类型** | 网线、光纤、电源线、串口线等 |
| **线缆编号** | 物理线缆的唯一标识 |
| **长度（米）** | 线缆物理长度 |
| **备注** | 走线路由、标签等补充信息 |

### 创建接线

1. 进入设备详情页 → 「接线管理」标签页
2. 点击「新增接线」按钮
3. 选择源设备与源端口
4. 选择目标设备与目标端口
5. 填写线缆编号、类型、长度等信息
6. 点击「确定」保存

### 可视化追踪

接线管理支持对特定设备或端口进行连接追踪：

- **正向追踪**：从指定端口出发，查看其链接到哪些目标设备端口
- **反向追踪**：查看哪些设备的端口链接到当前端口
- **全链路追踪**：从源设备出发，沿连接关系逐跳展示完整链路路径

追踪结果以拓扑图形式展示，支持高亮显示追踪路径。

### 清单导出

1. 在接线管理页面设置筛选条件（按机房、机柜、设备、线缆类型等）
2. 点击「导出接线清单」按钮
3. 选择导出格式（Excel/CSV）
4. 导出的清单包含：源设备、源端口、目标设备、目标端口、线缆编号、线缆类型、长度等信息

## 网络拓扑

网络拓扑以可视化方式展示数据中心内设备的网络连接关系，基于 ReactFlow 引擎实现。

### 拓扑展示

- **自动布局**：根据设备间的连接关系自动生成拓扑图布局
- **节点展示**：每个设备为一个节点，显示设备名称、类型、IP 地址等关键信息
- **连线展示**：设备间的连接关系以连线表示，不同线缆类型使用不同颜色和线型区分
- **状态标识**：设备节点根据运行状态显示对应颜色标识，异常设备高亮警示

### 交互操作

| 操作 | 方法 |
|------|------|
| **拖拽画布** | 鼠标左键拖拽空白区域平移视图 |
| **缩放画布** | 鼠标滚轮缩放 |
| **查看设备详情** | 点击设备节点弹出详情面板 |
| **查看连接详情** | 点击连接线查看连接信息 |
| **框选节点** | 按住 Shift 拖拽鼠标框选多个设备 |
| **导出拓扑图** | 点击导出按钮保存为图片 |

### 拓扑过滤

支持按以下条件过滤拓扑展示内容：

- 按机房/机柜范围过滤
- 按设备类型过滤（服务器、交换机、存储等）
- 按设备状态过滤
- 按连接类型过滤

## 字段管理

字段管理用于自定义设备属性，满足不同场景下的资产管理需求。

### 支持字段类型

| 字段类型 | 说明 |
|----------|------|
| **文本** | 单行文本输入 |
| **数字** | 数值输入 |
| **日期** | 日期选择器 |
| **日期时间** | 日期+时间选择器 |
| **下拉选择** | 单选下拉框，可自定义选项列表 |
| **多选** | 多选下拉框，可同时选择多个选项 |
| **多行文本** | 长文本输入 |
| **开关** | 布尔值开关 |
| **人员选择** | 从系统用户中选择 |

### 管理自定义字段

1. 进入「设备管理」页面 → 「字段管理」标签页
2. 点击「新增字段」按钮
3. 填写字段信息：字段名称、字段类型、是否必填、排序、默认值、选项列表（下拉/多选类型需要）
4. 保存后，自定义字段将自动出现在设备新增/编辑表单中

### 字段排序与可见性

- **排序**：拖拽调整自定义字段在表单中的排列顺序
- **可见性**：可设置字段在列表和表单中的显示/隐藏
- **必填校验**：设置为必填的字段在保存时会进行非空校验

# 资产管理

## 设备管理

设备管理是系统的核心功能，提供设备全生命周期管理。

### 设备状态流转

```
采购中 → 待安装 → 运行中 ⇄ 维护中 → 已报废
```

| 状态 | 说明 |
|------|------|
| **采购中** | 设备已采购，尚未到货 |
| **待安装** | 设备到货，等待安装 |
| **运行中** | 设备正常运行 |
| **维护中** | 设备正在维护 |
| **已报废** | 设备已报废处理 |

### 新增设备

1. 进入「设备管理」页面
2. 点击「新增设备」按钮
3. 填写设备信息：名称、类型、型号、序列号、品牌、所属机房、机柜、U位、状态等
4. 点击「确定」保存

### 批量导入

1. 点击「导入」按钮
2. 下载导入模板（Excel/CSV格式）
3. 按模板格式填写设备信息
4. 上传文件，系统自动解析并导入

### 设备导出

1. 选择需要导出的设备（可多选）
2. 点击「导出」按钮
3. 选择导出格式（Excel/CSV）
4. 文件自动下载

## 端口管理

端口管理用于配置设备的网络端口信息。

### 端口类型

| 类型 | 说明 |
|------|------|
| **网口** | 以太网接口（RJ45） |
| **光口** | 光纤接口（SFP/SFP+） |
| **管理口** | 带外管理接口 |
| **其他** | 其他类型接口 |

## 接线管理

接线管理用于记录机柜间的线缆连接关系，支持线缆追踪和清单导出。

## 字段管理

字段管理用于自定义设备属性，支持文本、数字、日期、下拉选择、多行文本等字段类型。
